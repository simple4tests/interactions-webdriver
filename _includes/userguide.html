Wait catchTimeoutException() By default the engine wait for a condition that must be realized.
If not, there is a failure and the test stops with failed status.
Some times we need to wait for a condition that may be realized (or not).
This function configure the wait engine to wait for a condition to be realized and if not relalized to continue the test.
  • wait.elementToBePresent(By by) :
    if the condition is not realized the test stop with status failed
  • wait.catchTimeoutException().elementToBePresent(By by) :
    if the condition is not realized the test continue
The call of this function affects only the first following wait.
Boolean elementToBePresent(By by) Wait for the element to exist in the page
public <T> T until(
    Function<WebDriver, T> expectedCondition) 		This functions wait for the expectedCondition to be relazided
