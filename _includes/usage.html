Usage

1. Add dependency on your pom.xml file {% highlight java %} io.github.simple4tests interactions-webdriver ... {% endhighlight %} 2. Instanciate a WebDriverInteractions object {% highlight jave %} WebDriver driver; ... driver = new ChromeDriver(); ... WebDriverInteractions ui = new WebDriverInteractions(driver); {% endhighlight %} 3. Just use it {% highlight jave %} ui.browser.navigateTo(...); ui.click(...); ui.set(...); ... {% endhighlight %}
